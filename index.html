3D Reconstruction from Stereo Feature Points

Developed a MATLAB-based pipeline for reconstructing 3D point clouds from stereo image pairs captured by calibrated cameras. Enabling geometric reconstruction of 3D structures from 2D observations.

Program: TWSIAM 2022 Summer Research Internship [Research Page]
Group: Image Processing Group
Research Topic: 3D Reconstruction with Two Calibrated Cameras [Results]
Supervisor: Chin-Tien Wu [Email]

VIEW MORE

Data: Two images captured from overlapping viewpoints
Pipeline:

• Camera Calibration: Obtained camera projection matrices (P) using MATLAB’s checkerboard-based calibration toolbox.
• Feature Detection: Used SIFT to detect keypoints and extract robust feature descriptors.
• Fundamental Matrix Estimation (F): Estimated F using the eight-point algorithm.
• Epipolar Constraint: Refined correspondences using epipolar geometry.

• Pose Recovery (E, R, t): Computed the essential matrix (E) from camera geometry and recovered the relative pose, including rotation (R) and translation (t), between the two views.
• Triangulation: Triangulated matched feature points using the recovered pose and projection matrices to reconstruct 3D coordinates.

Result: As shown in the demo video.

Skills: MATLAB, Camera Modeling, Computational Photography, 3D Reconstruction, Multi-view Stereo, Epipolar Geometry
